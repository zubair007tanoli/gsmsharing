Key Features
  • Automatic URL detection (HTTP, HTTPS, www)
  • Rich metadata extraction (Open Graph Protocol)
  • Compact link icons replace long URLs
  • Thumbnail images with lazy loading
  • Favicon and domain display
  • Hover effects and animations
  • Dark theme support
  • Mobile responsive design
  • Real-time processing for new comments
  • Batch API requests for performance
  • Graceful error handling
  • Security: SSRF protection, XSS prevention
Technical Implementation

Frontend:

  • JavaScript: Automatic URL detection with regex
  • MutationObserver: Watches for new comments
  • Batch API calls: 2-10 URLs per request
  • CSS: Modern flexbox/grid layout

Backend:

  • ASP.NET Core API Controller
  • LinkMetadataService: Open Graph extraction
  • HtmlAgilityPack: HTML parsing
  • 10-second timeout protection
